# Credit Card Financial Dashboard - Notes

## Overview
- Interactive dashboard built using Power BI
- Analyzes credit card transactions and customer behavior
- Focus on revenue, interest, and spending patterns

## Key Insights

### Revenue & Interest
- Tracks total revenue trends over time
- Monitors interest generation patterns

### Customer Segmentation
- Based on:
  - Age
  - Income
  - Marital Status
  - Job Profile
- Helps identify high-value customers

### Transaction Analysis
- Modes covered:
  - Swipe
  - Chip
  - Online
- Shows customer payment preferences

### Card Category Analysis
- Compares different card types
- Highlights spending behavior per category

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Modeling

## Dataset Includes
- Customer details (Income, Job, Age)
- Transaction data (Amount, Mode)
- Credit card categories
